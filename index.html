USA's Government Project Infographics

Two infographics have been designed and developed to represent data regarding USA's projects. '

Scatter Plot

The aim of the scatter plot is to allow analysis to see if there is any correlation between cost and schedule varience when looking at the project size. The circles represent the different levels of cost, with the locations on the axis showing the variations. There is basic zooming into areas, and allows selections to highlight different departments to see and compare how they are performing.

Sunburst

The sunburst is useful for determining how the project budget is allocated. It clearly shows how it is divided, by department, investment area, and project. It's aim is to show instantly how the project budget is divided, and the internal seperations of each department. It allows dynamic interaction, being able to filter down by department and suchlike to allow in depth analysis.
